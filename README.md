# HTML-parser

Запуск в VCCode с помощью vcpkg:
```
    >CMake: Build
    ./build/ParserHTML esoteric
```

Запуск в папке "cmake_build" с помощью установленной заранее libcurl:
```
    cmake ..
    cmake --build .
    ./Debug/ParserHTML esoteric
```
